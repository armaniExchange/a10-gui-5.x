# UI Auto Generator
## Usage
> npm run uigen

## Structure
```
adc
  virtual-server
    edit { v1, v2, v3...}
  wizard
    next1{ v1, v2}
    next2{ v1, v2}
  template...
```

## How to generate a page

### build a complex page
1. select a layout
2. drag component
3. edit properties
4. add some callback
5. save to ui schemas:
```
  {
    page URL(correspond directory path, router info),
    title,
    events...,
    page properties {
      components: {
        componentA: {
          properties,
          children: {

          }
        },
        componentB: {...}
      }
    }
  }
  6. generate js files
```

### simple page
1. CM json schemas to ui schemas
2. GUI IDE show the ui generated by ui shemas
3. Edit ui schemas by GUI IDE
4. Save to new version ui schemas
